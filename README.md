# 🌆 City of Joy Tribute

A tribute webpage dedicated to the beauty, culture, and spirit of Kolkata — the "City of Joy".

---

## 📌 About the Project

This project is a simple yet meaningful tribute page that highlights:
- The culture and heritage of Kolkata
- Iconic landmarks
- The emotional connection people have with the city

Built as part of my web development learning journey.
